# <a name="changelog-for-power-bi-report-server"></a>Журнал изменений для сервера отчетов Power BI

Этот журнал изменений предназначен для сервера отчетов Power BI и содержит новые элементы, а также исправления ошибок для каждой выпущенной сборки.

Дополнительные сведения о новых возможностях см. в статье [Новые возможности сервера отчетов Power BI](whats-new.md). 

## <a name="may-2019"></a>Мая 2019 г.

- **Сервер отчетов Power BI**          
    - *Версия выпуска 1.5.7074.36177 (сборка 15.0.1102.371): 21 мая 2019 г.*
        - Исправленные ошибки
            - Отчеты с разбивкой на страницы
                - Исправление всегда включать внедрение шрифтов pdf.
                - Исправление для установки файлов cookie, отправленных по протоколу https, как Secure
                - Исправление для проблемы, связанные с pop ИБП из-за ошибки сценария
                - Исправление для отображения с помощью мобильного приложения на телефонах с Android
                - Исправление ошибки, Навигатор по времени мобильного отчета для отображения номера недель правильный независимо от начала финансового года
                - Добавлены «RestrictedResourceMimeTypeForUpload» заблокированный устанавливаемое свойство для "Администраторы" указать mime-типы
         - Функции
            - Добавление поддержки для надежных визуальных элементов в PBIRS

- **Power BI Desktop (оптимизировано для решения "Сервер отчетов Power BI")** .
    - *Версия: 2.69.5467.1801 (мая 2019 г.), выпуска: 21 мая 2019 г.*
        - Исправленные ошибки
            - Исправление, чтобы избежать перезапуска учетных данных во время отправки pbix-ФАЙЛ в PBIRS
            - Открытие документов с #, в имени файла исправления
            - Добавлена ссылка проще для переходов назад на окно выбора PBIRS
            - Исправление для режима высокой контрастности в PBIRS для отображения кнопки "Назад", отображать предупреждающие сообщения visual.
            - Область выделения, исправлены ошибки пользовательского интерфейса на основе холста масштабирования.
