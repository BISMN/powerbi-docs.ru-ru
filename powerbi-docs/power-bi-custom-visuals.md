# <a name="custom-visuals-in-power-bi"></a>Пользовательские визуальные элементы в Power BI

При создании или редактировании отчета Power BI, можно использовать множество различных типов визуальных элементов. Значки для эти визуальные элементы отображаются в **визуализации** области. Эти визуальные элементы предварительно упаковываются, при загрузке [Power BI Desktop](https://powerbi.microsoft.com/desktop/) или откройте [службе Power BI](https://app.powerbi.com).

![визуализации](media/power-bi-custom-visuals/power-bi-visualizations.png)

Тем не менее можно не только этот набор визуальных элементов. Если выбрать многоточие (...) в нижней, другой источник визуальных элементов отчетов становится доступным -*пользовательские визуальные элементы*.

Разработчики создают пользовательские визуальные элементы, с помощью пакета SDK для пользовательских визуальных элементов. Эти визуальные элементы позволяют корпоративным пользователям просматривать свои данные в виде, что наилучшим образом соответствует свой бизнес. Авторы отчетов можно импортировать файлы пользовательских визуальных элементов в свои отчеты и использовать их, как и другие визуальные элементы Power BI. Пользовательские визуальные элементы являются привилегированными в Power BI и может фильтроваться, выделенные, измененный, общая и т. д.

Пользовательские визуальные элементы развертываются с тремя способами:

* Файлы пользовательских визуальных элементов
* Визуальные элементы организации
* Визуальные элементы из Marketplace

## <a name="custom-visual-files"></a>Файлы пользовательских визуальных элементов

Пользовательские визуальные элементы представляют собой пакеты с кодом для отображения данных, предназначенных для этих. Любой пользователь может создать пользовательский визуальный элемент и его в пакет как одиночный `.pbiviz` файл, который затем можно импортировать в отчет Power BI.

> [!WARNING]
> Пользовательский визуальный элемент может содержать код безопасности или конфиденциальности. Убедитесь, что вы доверяете автору и источнику пользовательских visual прежде чем импортировать их в отчет.

## <a name="organizational-visuals"></a>Визуальные элементы организации

Администраторы Power BI утверждение и развертывание пользовательских визуальных элементов в своей организации, который авторы отчетов могут легко обнаружить, обновлять и использовать. Администраторы могут легко управлять (например, обновление версии, включить или отключить) эти визуальные элементы.

 [Дополнительные сведения о визуальных элементов организации](power-bi-custom-visuals-organization.md).

## <a name="marketplace-visuals"></a>Визуальные элементы из Marketplace

Члены сообщества Microsoft свои пользовательские визуальные элементы для общих преимуществ, влияющая и публикации их [AppSource](https://appsource.microsoft.com/marketplace/apps?product=power-bi-visuals) marketplace. Можно загрузить эти визуальные элементы добавлены в отчетах Power BI. Корпорация Майкрософт протестированы и утверждены эти пользовательские визуальные элементы для функциональности и качества.

Что такое [AppSource](developer/office-store.md)? Это место, где можно найти приложения, надстройки и расширения для программного обеспечения Майкрософт. [AppSource](https://appsource.microsoft.com/) позволяет миллионам пользователей продуктов, как Office 365, Azure, Dynamics 365, Cortana и Power BI, чтобы решения, которые помогают им выполнять работу более эффективно, insightfully и визуализации больше, чем раньше.

### <a name="certified-visuals"></a>Сертифицированные визуальные элементы

Power BI сертифицированные визуальные элементы являются визуальные элементы из marketplace, которые прошли проверку дополнительных строгих качества и поддерживаются в дополнительных сценариях, таких как [подписок по электронной почте](https://docs.microsoft.com/power-bi/service-report-subscribe), и [Экспорт в PowerPoint](https://docs.microsoft.com/power-bi/service-publish-to-powerpoint).
Просмотреть список сертифицированных пользовательских визуальных элементов или отправить свои визуальные элементы можно [здесь](https://docs.microsoft.com/power-bi/power-bi-custom-visuals-certified).

Вы являетесь веб-разработчиком и заинтересованы в том, чтобы создавать собственные визуальные элементы и добавлять их в AppSource? См. в разделе [Разработка пользовательских визуальных элементов Power BI](developer/custom-visual-develop-tutorial.md) и узнайте, как [публикация настраиваемых визуальных элементов в AppSource](https://docs.microsoft.com/power-bi/developer/office-store).

### <a name="import-a-custom-visual-from-a-file"></a>Импорт настраиваемого визуального элемента из файла

1. Щелкните многоточие в нижней части **визуализации** области.

    ![Визуализации 2](media/power-bi-custom-visuals/power-bi-visualizations2.png)

2. В раскрывающемся списке выберите **Импортировать из файла**.

    ![Импорт из файла](media/power-bi-custom-visuals/power-bi-custom-visual-import-from-file.png)

3. В меню "Открыть файл" выберите `.pbiviz` файл, который требуется импортировать, а затем выберите **откройте**. Значок пользовательского визуального элемента добавляется в нижнюю часть вашей **визуализации** области и теперь доступен для использования в отчете.

    ![CV-файл импортирован](media/power-bi-custom-visuals/power-bi-custom-visual-imported.png)

### <a name="import-organizational-visuals"></a>Импорт визуальных элементов организации

1. Щелкните многоточие в нижней части **визуализации** области.

    ![Визуальный элемент организации 1](media/power-bi-custom-visuals/power-bi-visual-org-01.png)

2. В раскрывающемся списке выберите **Импорт из магазина**.

    ![Визуальный элемент организации 2](media/power-bi-custom-visuals/power-bi-visual-org-02.png)

3. Выберите **MY ORGANIZATION** (Моя организация) в верхней части меню вкладки.

    ![Визуальный элемент организации 3](media/power-bi-custom-visuals/power-bi-visual-org-03.png)

4. Прокрутите список и найдите визуальный элемент для импорта.

    ![Визуальный элемент организации 4](media/power-bi-custom-visuals/power-bi-visual-org-04.png)

5. Выберите **добавить** импорт пользовательского визуального элемента. Значок добавляется в нижнюю часть вашей **визуализации** области и теперь доступен для использования в отчете.

    ![Визуальный элемент организации 5](media/power-bi-custom-visuals/power-bi-visual-org-05.png)

## <a name="download-or-import-custom-visuals-from-microsoft-appsource"></a>Скачать или импортировать настраиваемые визуальные элементы из Microsoft AppSource

Существует два варианта загрузки и импорта пользовательских визуальных элементов: из Power BI и из [веб-сайта AppSource](https://appsource.microsoft.com/).

### <a name="import-custom-visuals-from-within-power-bi"></a>Импорт настраиваемых визуальных элементов из Power BI

1. Щелкните многоточие в нижней части **визуализации** области.

    ![Визуализации 2](media/power-bi-custom-visuals/power-bi-visualizations2.png)

2. В раскрывающемся списке выберите **Импорт из магазина**.

    ![Визуальный элемент организации 2](media/power-bi-custom-visuals/power-bi-visual-org-02.png)

3. Прокрутите список и найдите визуальный элемент для импорта.

    ![Импорт визуального элемента](media/power-bi-custom-visuals/power-bi-import-visual.png)

4. Чтобы получить дополнительные сведения об одном из визуальных элементов, выделите и выберите его.

    ![Выбрать](media/power-bi-custom-visuals/power-bi-select.png)

5. На странице сведений могут быть представлены снимки экрана, видео, подробное описание и многое другое.

    ![Synoptic](media/power-bi-custom-visuals/power-bi-synoptic.png)

6. Прокрутите вниз, чтобы просмотреть обзоры.

    ![Отзывы](media/power-bi-custom-visuals/power-bi-reviews.png)

7. Выберите **добавить** импорт пользовательского визуального элемента. Значок добавляется в нижнюю часть вашей **визуализации** области и теперь доступен для использования в отчете.

    ![Визуальный элемент импортирован](media/power-bi-custom-visuals/power-bi-custom-visual-imported.png)

### <a name="download-and-import-custom-visuals-from-microsoft-appsource"></a>Скачивание и импорт настраиваемых визуальных элементов из Microsoft AppSource

1. Войдите в [Microsoft AppSource](https://appsource.microsoft.com) и выберите вкладку **Приложения**.

    ![AppSource](media/power-bi-custom-visuals/power-bi-appsource-apps.png)

2. Перейдите на [страницу результатов с приложениями](https://appsource.microsoft.com/marketplace/apps), где показаны лучшие представители каждой категории, включая *приложения Power BI*. Мы ищем настраиваемые визуальные элементы, так что давайте выберите **визуальных элементов Power BI** в списке навигации слева, чтобы сузить результаты.

    ![Визуальные элементы AppSource](media/power-bi-custom-visuals/power-bi-appsource-visuals.png)

3. В AppSource отобразится плитка для каждого настраиваемого визуального элемента.  Каждая плитка содержит пользовательские снимок визуального краткое описание и ссылку для скачивания. Выберите плитку, чтобы получить дополнительные сведения.

    ![Выбор пользовательского визуального элемента](media/power-bi-custom-visuals/powerbi-custom-select-visual.png)

4. На странице сведений могут быть представлены снимки экрана, видео, подробное описание и многое другое. Выберите **получить сейчас** для загрузки пользовательского визуального элемента и согласитесь с условиями использования.

    ![Получение AppSource](media/power-bi-custom-visuals/power-bi-appsource-get.png)

5. Перейдите по этой ссылке для скачивания настраиваемого визуального элемента.

    ![Скачать](media/power-bi-custom-visuals/powerbi-custom-download.png)

    На странице загрузки также содержит инструкции о том, как импортировать пользовательский визуальный элемент в Power BI Desktop и службе Power BI.

    Вы также можете скачать пример отчета, в котором используется этот пользовательский визуальный элемент и демонстрируются его возможности.

    ![Пример](media/power-bi-custom-visuals/powerbi-custom-try-sample.png)

6. Сохранить `.pbiviz` и затем откройте Power BI.

7. Импорт `.pbiviz` файла в отчет. (См. дополнительные сведения в разделе [Импорт настраиваемого визуального элемента из файла](#import-a-custom-visual-from-a-file) выше.)

## <a name="considerations-and-limitations"></a>Рекомендации и ограничения

* При импорте пользовательский визуальный элемент добавляется в определенный отчет. Если вы хотите использовать этот визуальный элемент в другом отчете, необходимо также импортировать его в этот отчет. При сохранении отчета с пользовательским визуальным элементом с помощью параметра **Сохранить как** вместе с новым отчетом сохраняется копия пользовательского визуального элемента.

* Если вы не видите **визуализации** области, это значит, что у вас нет отчетов изменение разрешений.  Чтобы добавить настраиваемые визуальные элементы в отчеты, необходимо обладать правами на внесение изменений. Вы не можете добавлять их в отчеты, к которым предоставлен общий доступ.
